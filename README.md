# KAIFreeAudiencesBot

<img src=".assets/logo.png" align="right" width="150" />

Телеграм бот для отображения свободных аудиторий в зданиях КАИ.
## Stack
* ASP.NET 6.0, C# 8.0
* sqlite3
* python 3.6+
* nginx
* ngrok(for debug only)
* docker-compose
